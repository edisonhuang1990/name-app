# 周易起名网 (Naming App)

这是一个基于周易八字五行理论的宝宝起名网站。网站风格简约，旨在为父母提供既符合传统文化又好听的名字选项。

## 功能特点

*   **八字排盘**：根据出生日期和时辰自动计算四柱（年、月、日、时）干支。
*   **五行分析**：统计五行（金、木、水、火、土）数量，自动识别缺少的元素（喜用神）。
*   **智能起名**：根据五行缺失情况，从精选字库中推荐名字。
*   **卦象解读**：随机生成周易卦象，赋予名字更深层的寓意。
*   **简约设计**：采用宋体字和留白设计，阅读舒适。

## 项目结构

*   `index.html`: 网站主页
*   `style.css`: 样式文件
*   `script.js`: 核心逻辑（八字计算、起名算法）
*   `data.js`: 数据文件（汉字库、卦象库）
*   `server.js`: 本地开发服务器

## 如何运行

1.  确保已安装 Node.js。
2.  在项目根目录下运行：
    ```bash
    node server.js
    ```
3.  打开浏览器访问：[http://localhost:3000](http://localhost:3000)

## 技术栈

*   HTML5 / CSS3
*   Vanilla JavaScript (ES6+)
*   Node.js (用于本地服务)
